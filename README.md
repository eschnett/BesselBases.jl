# BesselBases.jl

Expand functions into [spherical Bessel
functions](https://en.wikipedia.org/wiki/Bessel_function#Spherical_Bessel_functions).

[![CI](https://github.com/eschnett/BesselBases.jl/actions/workflows/CI.yml/badge.svg)](https://github.com/eschnett/BesselBases.jl/actions/workflows/CI.yml)

## Overview

Spherical Bessel functions are useful e.g. when solving partial
differential equations in a spherical domain. They go together well
with [spin-weighted spherical
harmonics](https://en.wikipedia.org/wiki/Spin-weighted_spherical_harmonics).
